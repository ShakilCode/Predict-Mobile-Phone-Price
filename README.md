# 📱 Predict Mobile Phone Price

A Machine Learning web application that predicts the price range of a mobile phone based on its hardware specifications.

---

## 🚀 Project Overview

This project builds a classification system that predicts whether a mobile phone belongs to:

- Low Cost
- Medium Cost
- High Cost
- Very High Cost

The model is trained on real mobile hardware features and deployed using **Streamlit** for an interactive UI.

---

## 🎯 Objective

To develop an intelligent system that:

- Analyzes mobile phone specifications  
- Learns patterns from historical data  
- Predicts the price category accurately  
- Provides an easy-to-use web interface  

---

## 🧠 Machine Learning Model

- Algorithm: **Logistic Regression**
- Problem Type: **Multi-class Classification**
- Preprocessing: **StandardScaler**
- Input Features: **20**
- Output Classes: **4**
